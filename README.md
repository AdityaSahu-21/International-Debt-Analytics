# International-Debt-Analytics
A data-driven analytics platform to collect, process, analyze, and visualize international external debt data across countries using Python, SQL, and Power BI.

The International Debt Analytics Platform is a data analytics project designed 
to transform large and complex international debt datasets into meaningful insights.

The platform performs country-wise and region-wise debt analysis, evaluates debt 
sustainability through key indicators such as Debt-to-GDP and Debt Service Ratios, 
and presents findings through interactive dashboards and comparative reports.

Data Source  : World Bank International Debt Statistics (IDS)

Technologies : Python (Pandas, NumPy, Matplotlib), SQL, Power BI, MS Excel
