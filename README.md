# 🌍 International Debt Analytics Platform

> A data-driven analytics platform to collect, process, analyze, and visualize international external debt data across countries using Python, SQL, and Power BI.

---

## 📌 About the Project

The **International Debt Analytics Platform** is designed to transform large and complex international debt datasets into meaningful insights. The platform addresses key challenges faced by analysts and researchers when dealing with multi-dimensional debt data.

By leveraging data analytics techniques, statistical indicators, and interactive visualization tools, the platform enables users to:
- Examine country-wise and region-wise debt trends
- Assess debt sustainability
- Compare repayment capacity across economies

---

## 🎯 Objectives

- Analyze international external debt patterns using structured datasets
- Provide country-wise and region-wise debt indicators for comparison
- Build interactive dashboards for financial statistics
- Evaluate debt sustainability through key performance metrics (Debt-to-GDP, Debt Service Ratios)
- Support policy and research use-cases in global financial monitoring

## 📊 Key Features

- **Country-level debt profiling** — aggregation and analysis per nation
- **Trend analysis** — time-series exploration of borrowing growth
- **Debt sustainability metrics** — Debt-to-GDP and Debt Service Ratios
- **Creditor distribution insights** — multilateral, bilateral, and private creditors
- **Interactive dashboards** — comparative study across regions
- **Reporting module** — exportable reports for analysts and researchers

## 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/AdityaSahu-21/International-Debt-Analytics.git
```

2. Install required Python libraries:
```bash
pip install pandas numpy matplotlib jupyter
```

3. Open the Jupyter Notebook:
```bash
jupyter notebook ida.ipynb
```

4. For dashboard — open the Power BI `.pbix` file in **Power BI Desktop**
